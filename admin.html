📦 Panduan Commit Manual

  1. 1. Salin JSON yang telah diperbarui (klik tombol 📋 Salin JSON di form)
  2. 2. Buka repositori GitHub Anda di browser
  3. 3. Navigasi ke file artikel.json
  4. 4. Klik ikon ✏️ Edit (pensil)
  5. 5. Paste konten JSON yang telah disalin
  6. 6. Scroll ke bawah, isi commit message, lalu klik Commit changes
  7. 7. Tunggu GitHub Actions deploy otomatis (~1-2 menit)

💡 Tips:

  • Gunakan JSONLint untuk validasi format
  • Backup artikel.json sebelum edit manual
  • Untuk workflow otomatis, setup GitHub Actions dengan workflow_dispatch
